Aplicación de prueba para testear el consumo desde APIs

Esta aplicaión será utilizada para validar las conexiones de apis, para ello se utiliza el api abierto de REST countries, el cual devuelve información de paises dependiento del api utilizado. Para nuestro caso utilizaremos el api https://restcountries.eu/rest/v2/lang/[lang] que devuelve los paises que hablan el idioma especificado en [lang] (es, en, fr...etc)
